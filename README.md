# eFileCounter
Real-time file counter designed for production box tracking and control.

## Overview

eFileCounter was developed within a test department environment to validate tested and packaged circuits.  
It allows defining counting targets, provides real-time visual feedback, and dynamically adapts the counter during operation.

The tool is available in two versions:
- **Basilisk**: for modern systems  
- **Pangolin**: for deprecated systems
